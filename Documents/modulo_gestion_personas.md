# Guía Detallada: Gestión de Clientes y Usuarios (2026)
## Matteo Salvatore - Personas y Seguridad

Este documento describe el funcionamiento de los módulos de gestión humana y seguridad del ecosistema digital de **Matteo Salvatore**.

---

## 1. Módulo de Clientes (Base de Datos Unificada)
El sistema rastrea automáticamente a cada persona que interactúa con la tienda, unificando la experiencia física (POS) y digital (E-commerce):
- **Registro Automático**: Se crea un perfil de cliente tras la primera compra (incluso como invitado) basándose en su correo electrónico.
- **Historial de Pedidos Dinámico**: Los administradores pueden ver todas las compras pasadas del cliente desde una única vista.
- **Segmentación Inteligente**: Filtros para identificar clientes recurrentes, compradores estacionales o clientes que compran exclusivamente en tiendas físicas.
- **Privacidad y Datos**: Cumple con altos estándares de seguridad y protección de datos personales.

---

## 2. Diferenciación: Clientes vs Invitados
- **Para Invitados**: El sistema optimiza el flujo de compra eliminando la necesidad de crear una contraseña, pero manteniendo la vinculación de sus datos para futuros seguimientos.
- **Para Usuarios Registrados**: Acceso a su perfil, agenda de direcciones guardadas y estado de sus pedidos actuales o históricos.

---

## 3. Gestión de Usuarios Administrativos
Control de acceso corporativo para el equipo interno de la marca:
- **Perfiles de Acceso**: Diferenciación clara entre administradores con permisos totales y personal de ventas/almacén (que operan principalmente el POS e Inventario).
- **Seguridad Robusta**: Autenticación protegida para el ingreso al Panel de Control (Admin Dashboard).
- **Control de Logs**: El sistema registra qué usuario realizó cambios críticos (como despachar un pedido o ajustar manualmente el stock) para asegurar la transparencia operativa.

---

## 4. Gestión de Direcciones y Perfiles
- **Múltiples Direcciones**: Capacidad para guardar diferentes direcciones de envío para un mismo cliente.
- **Internacionalización de Datos**: Soporte para formatos de dirección tanto en español como en inglés, garantizando que los couriers reciban la información de despacho sin errores gramaticales.

---
*Este módulo garantiza que cada interacción con Matteo Salvatore sea personalizada, segura y profesional.*
