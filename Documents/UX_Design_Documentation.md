# UX Design Documentation / Documentación de Diseño UX
## Matteo Salvatore E-commerce (Updated March 2026)

This document defines the user experience principles, design system, and legacy/modern user flows for the Matteo Salvatore platform.

Este documento define los principios de experiencia de usuario, el sistema de diseño y los flujos de usuario (legado y modernos) para la plataforma Matteo Salvatore.

---

## 1. Design Principles / Principios de Diseño

### Minimalist Luxury / Lujo Minimalista
- Clean interfaces with generous whitespace to emphasize product photography.
- Interfaces limpias con generoso espacio en blanco para enfatizar la fotografía del producto.

### Mobile First
- Fully responsive design optimized for seamless shopping on smartphones.
- Diseño totalmente responsivo optimizado para compras fluidas en smartphones.

---

## 2. Visual Identity / Identidad Visual

### Typography / Tipografía
- **Fonts**: Inter / Outfit (Modern sans-serif)
- **Scale**: Clear hierarchy between headings and bodies.

### Color Palette / Paleta de Colores
- **Primary**: Noir (#000000) - For emphasis and text.
- **Secondary**: Pearl White (#F9F9F9) - For backgrounds.
- **Accent**: Gold (#C5A059) - For specific call-to-actions.

---

## 3. Key User Flows / Flujos de Usuario Clave

### Product Discovery / Descubrimiento de Productos
1. **Dynamic Menu**: Browsing categories via the multi-level navigation bar.
2. **Visual Filtering**: Selecting sizes and colors on the Product Detail Page.
3. **Advanced Search**: Real-time filtering by product name, category, or style code (Código de Estilo).

### Checkout Flow / Flujo de Compra
- **Streamlined Process**: Integrated IGV (18%) calculation and secure Culqi payment processing.
- **Proceso Simplificado**: Cálculo integrado de IGV (18%) y procesamiento seguro de pagos con Culqi.

---

## 4. Admin UX / UX del Panel Administrativo

### Menu Builder Interface
- **Intuitive Organization**: A two-column layout allowing admins to drag available categories into the menu structure.
- **Organización Intuitiva**: Un diseño de dos columnas que permite a los administradores arrastrar categorías disponibles a la estructura del menú.

### Product Management / Gestión de Productos (New 2026)
- **Efficient Identification**: Integrated "Código de Estilo" column in the product table for ultra-fast logistics tracking.
- **Identificación Eficiente**: Columna de "Código de Estilo" integrada en la tabla de productos para un rastreo logístico ultrarrápido.
- **Intelligent Search**: Real-time, multi-field search engine (Name, Style Code, Category) to manage large catalogs with zero friction.
- **Búsqueda Inteligente**: Motor de búsqueda en tiempo real (Nombre, Código de Estilo, Categoría) para gestionar catálogos grandes sin fricciones.

### Elite AI Content Generation / Generación de Contenido Elite AI
- **Automated Luxury Assets**: One-click generation of SEO-optimized titles, museum-grade descriptions, and social media kits (Instagram/TikTok).
- **Activos de Lujo Automatizados**: Generación en un clic de títulos optimizados para SEO, descripciones de alta gama y kits para redes sociales (Instagram/TikTok).
- **Translation Continuity**: Automatic handling of ES/EN content with cultural nuance for global dominance.
- **Continuidad de Traducción**: Manejo automático de contenido ES/EN con matices culturales para dominancia global.
